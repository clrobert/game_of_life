###  Game of Life
A reproduction of Conway's Game of Life written using OpenGL.

The view is kinda buggy on windows platforms.

Left-click to set up live cells.

Keys:
S: Begin simulation
E: Stop simulation
-: Slow simulation
=: Speed simulation
